Workshop 2 - Clean Code

Overview

In this workshop you’ll be given simple project with regressions tests and be asked to refactor the code to make it more maintainable and to add some new functionality.

The Aim

We’ll code in pairs and then compare our solutions, to see how people have discussing the pros and cons of each solution.

The aim of the workshop is to discuss:

  • how we write maintainable code other will understand
  • what factors are important in writing clean code
  • how should we structure our code to make it easily understood
  • the pros and cons of different solutions
  • how do we balance the time spent adding new features with time spent refactoring

Preparation

If you wish to attend please bring a laptop and download the source from here.

There code has been translated into several languages so you can do the workshop in a language you’re familiar with. Please pick one version and get the app and tests running before you start the workshop. All the versions are identical.
